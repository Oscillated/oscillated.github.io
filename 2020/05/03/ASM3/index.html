C语言查看该数内存存储情况：

反向输出的原因：PC以小端格式储存，因此输出结果为小端表示。内存中无论是整数还是小数，用多个字节表示时，一定是小端保存法。小数在内存中也以小段方式保存，与整数相同。

算术运算

  • add 加

    add 寄存器, 寄存器 如 add ax, bx

    add 寄存器, 常数 如 add ax, 2

    add 寄存器,变量 如 add ax, ds:[1000h]

    add 变量, 寄存器 如 add ds:[1000h], ax

    add 变量, 常数 如 add word ptr ds:[1000h], 2

    • 小数加法fadd
    • 不能add 变量, 变量，因为硬件cpu不支持同时操作两个内存变量。

  • 汇编中常数没有宽度 1可以为8位、16位、32位

  • 确定运算时变量的宽度，引入ptr。

    • word ptr ds:[1000h] 表示 ds:1000h所指向的对象是一个word类型的变量，相当于*(short int *)(ds:1000h) 两个字节

    • byte ptr 相当于char* 一个字节

    • dword ptr 相当于long int* 四个字节
