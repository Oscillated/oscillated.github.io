集成运放设计基础

跟随器电路

  • 用作隔离级
    • 增益为1
    • 输入阻抗高（相当于对前级开路）、输出阻抗低（相当于对后级的电压源、内阻近乎零）
    • 使前后级电路之间互不干扰
    • 反馈回路有保护电阻的适合于输入端内已并接两组保护二极管的运放，避免波形出现奇异高次谐波

积分、微分电路

积分器

  • 无源的：RC低通
    • 无增益、有衰减
  • 有源的：运放结构的反馈回路加上电容
    • 输出电压与输入电压的积分成正比
    • 线性良好的扫描电路

微分器

  • 无源的：RC高通
  • 有源的：有源积分器内将R和C调换位置
    • 输出电压与输入电压的微分成正比

有源更接近实际情况，更加精确

精密半波整流电路

正半周去除，负半周反向

有源滤波器Active Filter —— 使用运放

无源滤波——仅使用电阻电感电容

有源低通

反相型、同相型 电容并反馈电阻

有源高通

电容串传入位置电阻

带通、带阻

单电源应用电路

一般双电源供电，以便在静态时输出端直流电位为零。

交流放大器中因为输入输出都可加隔直电容，为了供电方便也可改用单电源供电。

放大微弱信号一般还是要用双电源。

“只能用0-5V电源”

单电源供电

为了平衡，要求运放输出端的直流电位为$\frac{1}{2}V_{cc}$（双电源供电时要求运放输出端直流电位为0），所以反向端的直流电位也为$\frac{1}{2}V_{cc}$，为使运放正常工作，同相端也应加$\frac{1}{2}V_{cc}$直流电位，以保证集成运放的两个输入端有相同的直流电位

比较器电路

过零比较器

齐纳二极管、反相输入，运放工作在非线性区

$V_i>0，V_o$为负，$V_i<0，V_o$为正

单限比较器

同相输入，与参考电压$U_r$比较，需要上拉电阻才能输出高电平

OC - 集电极开路

迟滞比较器

相当于加正反馈的单限比较器

反相输入

克服了单限比较器在$V_r$附近剧烈抖动的缺点

施密特Schmitt触发器

类比磁滞回线，跳变电压值附近干扰不超过$\Delta V$，输出电压就是稳定的

同相输入：$V_i$超过上界后，输出为较大值

反相输入：$V_i$超过上界后，输出为较小值

缺点是分辨率降低（差别小于$\Delta V$的两个输入电压值）

计算迟滞比较器的输入上下限，已知$V_{OH},V_{OL}$

  • 重点是临界条件$V_+=V_-$

没有稳压二极管时，输出$V_0=运放电源电压范围$对于R-R轨到轨运放

方波三角波发生器 = 积分器+同相输入迟滞比较器

第一级对第二级输出的方波进行积分得到三角波，第二级对第一级输出的三角波进行迟滞比较得到方波 Multisim仿真

电压电流转换器

电压传输会有衰减，那么可以转换成电流来传。

在信号源附近先将信号电压放大，然后将放大了的电压转换成一个与其成正比的恒流源，再传送到负载两端

霍兰德恒流源

既有负反馈又有正反馈，负载电流与负载值无关，满足了横流要求

数据放大器

采集微弱信号（小差模电压、一定共模电压输出、输出电流较小）

要求高增益、高共模抑制比、高精度（失调漂移小）、高速（频带宽、摆率大）

三运放仪表放大器

运放差动组态，差分输入比例运算电路

输入电阻极高、共模抑制比极高、增益调节方便，放大差模电压、抵消共模电压、放大器对被测电路影响很小

可编程数据放大器 BB3606

隔离放大器

输入输出之间没有直接电路耦合（无公共地端）

变压器耦合隔离放大器、光电耦合隔离放大器

光电耦合隔离放大器

浮地（三角号）——不接大地，与大地无关绝缘的电压，用于隔离和避免干扰，参考地

运放器件datasheet阅读

输入阻抗 Impedance、Resistance

要考虑信源（被放大对象）的内阻

信源内阻忽略不计的误差为10% -> 选择10倍内阻以上输入阻抗

信源内阻忽略不计的误差为1% -> 选择100倍内阻以上输入阻抗

原则：信源内阻尽量忽略不计，选择足够大的运放输入阻抗（两者是串联关系）

带宽 Bandwidth、GBW、Frequency Response

GBW = Gain*Bandwidth 一般不是常数，要查手册

对电流反馈型CFB，根本不是常数

对电流反馈型VFB，不总是常数，查手册

共模抑制比 CMRR(Common-mode Rejection Ratio)

输入失调电压Input Offset Voltage

直流指标，对运放直流精度的影响最为直观

$V_{OS}$ 因同相端和反相端失配（工艺问题）而产生的输入级固有电压差，越小越好；有些运放的调零端可以使其为零，或使用另一个电压反向抵消

对高倍放大的精密电路重要

输入偏置电流 Input Bias Current

直流指标，对运放直流精度的影响最为直观

$I_B$ 输入级为了能正常工作而对输入晶体管进行偏置所需要的基极电流或栅极电流，越小越好

对高阻信源放大时非常重要，因为偏置电流乘以高阻产生很大误差电压

输入电压范围 Input Voltage、Range

将运放用于单电源供电时要特别注意

  1. VCC和GND电压差较小，输入不是轨到轨的话将限制输入电压的范围
  2. 输入信号小或信号中直流分量小的时候，就相当于输入逼近GND轨，如果不满足最小输入电压范围将不能正常工作

轨到轨

运放的输入电位通常要求高于负电源某一数值，而低于正电源某一数值。经过特殊设计的运放可以允许输入电位在从负电源到正电源的整个区间变化，甚至稍微高于正电源或稍微低于负电源也被允许。这种运放称为轨到轨。

为了防止出现削顶削底，只能让输入电位区间小于正负电压区间，空出部分称净空

输入电压范围正负电源轨都达不到的运放，不适合在单电源中应用

输入电压范围能达到负电源轨、但达不到正电源轨的运放，调节直流偏置即可应用在单电源中

噪声特性 Noise、Noise Density

在频率上积分后得到电压，再经过平方除以电阻，就可以得到功率

输出驱动能力 Current、Output

运放带负载的能力，作为最后一级时要考虑其驱动能力

current sourcing 拉电流：输出VCC时输出端对负载提供电流

current sinking 灌电流：输出GND时输出端从负载吸收电流

输出电压范围 Output Swing

在单电源供电的场合中选择轨到轨运放更方便使用

“swing from rail” 时 值就是净空

在最大输出幅度和供电电源轨之间必须有一定的裕量或净空，保证输出不被削顶削底，对输入也是一样（特殊工艺弥补三极管0.7V后，轨到轨的运放贵）

压摆率 Slew Rate

运放输出信号的最大斜率、表示运放输出大幅度信号的能力

结合GBW可以计算运放在输出最高频率信号时幅度的最大值

$s(t)=Asin(2\pi ft)$ ，$SR=A*2\pi f$不变时，幅值A（非峰峰值）越大，频率f就越小，需要A和f都大的应用场合，就需要选择压摆率大的运放

供电范围和静态电流 Power Supply、Quiescent Current

供电电压可以看出器件是否适合在单电源低电压下使用

静态电流可以看出器件是否适合在低功耗应用中使用
