MIT Linear Algebra 18.06 By Gilbert Strang Part1

Insights

线性变换

  • 左乘一个行向量/矩阵，所得结果相当于对原矩阵的行向量做线性组合。
  • 右乘一个列向量/矩阵，所得结果相当于对原矩阵的列向量做线性组合。

理解这两句话是1-4节课的核心和整个线性代数课程体系最最重要的基础。

解齐次和非齐次线性方程

列空间

$AX = b$ 的理解是：若有解，$A$经过右乘$X$，能找到线性组合（唯一或不唯一）表示出$b$，即b落在由线性组合定义的$A$的列空间中。

零空间

$AX = 0$ 的理解是：能够满足该等式方程的向量$X$在$A$的零空间中。

四个子空间

$m*n$的矩阵：

  • 列空间（$m$维空间的子空间）
  • 零空间（$n$维空间的子空间）
  • 行空间（$n$维空间的子空间）
  • 行空间的零空间（$m$维空间的子空间）。

列空间与行空间的零空间正交，零空间与行空间正交。

相互间的正交关系会在之后的投影中打通理解。

以上是5-8课的主要内容。

空间、矩阵、秩、维数、基

  • 秩是矩阵进行消元法后pivot的个数。
  • 列向量满秩，则列空间的基是线性无关且都在列空间内的$n$个$m$维向量，维数为秩$r$。
  • 行向量满秩，则行空间的基是线性无关且都在行空间内的$m$个$n$维向量，维数为秩$r$。
  • 零空间的维数是自由变量的个数$n-r$，满秩时零空间为零向量。
  • 矩阵对应秩，空间对应维数。

以上是一些主干知识外的内容。

投影矩阵与可逆阵的理解

投影方程：$p = P*b$

所有投影矩阵的性质：

$P^T = P$

$P^2 = P$

$A$为可逆方阵（满秩） -> $A$的列空间为整个空间 -> 用$A$能够表示$n$维空间中的任意一个向量 -> 要求$b$在$A$的列空间上的投影（方程$Ax=b$） -> $A$的列空间能够任意表示所以可以直接表示出$b$ -> $b$的投影就是$b$本身不变 -> 投影矩阵$P=I$

如果$A$不可逆，奇异，则找到的投影矩阵$P=A(A^TA)^{-1}A^T$，也可以剔除线性相关的向量，选列空间的基向量代入公式计算$P$。

$A$为向量，$P=\frac{aa^T}{a^Ta}$

投影矩阵的过程就是最小二乘法的原理，解$A^TA\hat{x} = A^Tb$
