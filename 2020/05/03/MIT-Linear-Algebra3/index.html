MIT Linear Algebra 18.06 By Gilbert Strang 3

Insights

正定矩阵

  • 如果有一个正定矩阵A，那么A的逆一定也是正定的。因为A的逆的特征值都是原来的倒数，符号相同

相似矩阵与特征值

  • 相似矩阵拥有相同的特征值，但是特征向量一般不相等，如果特征向量也相等那就是同一个矩阵了
  • 如果两个矩阵拥有n个相同的特征值，那么他们必然相似于某个对角矩阵。
  • 矩阵拥有相同特征值时不一定能对角化，只能相似于自己

在所有相似矩阵中抓出一个最典型，看起来最像相似矩阵但无法对角化的矩阵，称为若尔当标准型。将是否能对角化综合起来，提出若尔当块Jordan

  • 每个诺尔当块都要满足两个性质：主对角线元素完全相同（特征值完全相同），主对角线上方的次对角线元素全为1（如果有次对角线的话）
  • 如果一类矩阵可以化为相同的标准诺尔当型$J$（相同尺寸、相同数量），则这些矩阵全部是相似矩阵，都可以用$P^{-1}JP$来表示
  • 如果A有n个线性无关的特征向量。则A可以对角化，他的Jordan form 就是eigenvalue所组成的对角阵$J=\Lambda$
  • 如果A有相同的eigenvalue. 以及一些”丢失”的eigvector.那么它的Jordan form 应该是n-d个1在对角阵之上。

奇异值分解

将一个比较复杂的矩阵用更小更简单的几个子矩阵相乘来表示，这些小矩阵描述的都是矩阵的重要的特性，对可逆非可逆、方阵非方阵都适用。

正定矩阵的奇异值分解为$A=Q\Lambda Q^T$，用一个正交矩阵$Q$就可以使正定矩阵$A$进行奇异值分解，因此特征值分解是SVD的一种特殊形式。对于可对角化的矩阵来说，$A=S\Lambda S^{-1}$，但$S$不是正交矩阵。

可以把矩阵$A$看作一种线性变换操作，将其行空间中的每一个向量$v$，变换为列空间的向量$u$，即$u=Av$。类似地，我们也可以在行空间中找到一组正交基，通过矩阵$A$线性变换成列空间中的另一组正交基。

用到格拉姆-施密特正交化将一组基正交化，但某一组正交基经过线性变换后的向量未必正交，因此要找的正交基非常特殊，满足$AV=U$。进一步，在$V$中寻找标准正交向量使$U$中的向量标准化，从而使$U$和$V$标准正交：$AV=UΣ$，$Σ=diag(σ_i)$

在矩阵的四个子空间中选出合适的基，$v_1$到$v_r$是行空间的标准正交基，用零空间的标准正交基$v_{r+1}$到$v_n$补充完整，$u_1$到$u_r$是列空间的标准正交基，用左零空间的标准正交基$u_{r+1}$到$u_m$补充完整。$A$乘以每一个$v$对应一个$u$的方向，$Av_i=σ_iu_i$，$σ$称为奇异值，可将矩阵对角化$A=UΣV^{-1}=UΣV^T$。

求$U$、$V$的方法：$A^TA$、$AA^T$是特征值相同、特征向量不同的相似矩阵，将上式代入可以消去$U$、$V$其一，求出另一矩阵。

  • $A^TA=VΣ^TU^TUΣV^T=VΣ^2V^T$，$A^TA$是一个正定矩阵，它的特征向量标准正交组成$Q$（$V$），特征值是$σ^2$组成$Λ$。注意$σ$是$Av=σu$的伸缩因子，$σ^2$是$A^TA$的特征值。$σ$取$σ^2$的正平方根。
  • $AA^T=UΣV^TVΣ^TU^T=UΣ^2U^T$，$AA^T$是一个正定矩阵，它的特征向量标准正交组成$Q$（$U$），特征值是$σ^2$组成$Λ$。注意$σ$是$Av=σu$的伸缩因子，$σ2$是$AA^T$的特征值。
  • 步骤：求上述两矩阵的特征值和特征向量、标准化、写出$UΣV^T=A$
  • 要注意$V$与$U$的方向的方向是一致的，可能出现向量反向的问题，此时可以用过将$V$代入$AV=UΣ$来求出一致的$U$
  • 非满秩的$AA^T、A^TA$，还需要取零空间和左零空间内与行空间、列空间相垂直的向量补足
  • 长方矩阵可通过同样的过程进行奇异值分解

快速傅里叶变换FFT没懂P的作用

线性变换与矩阵

每个线性变换都对应于一个矩阵。矩阵变换的背后正是线性变换的概念。

一般的线性变换都满足加法不变性、数乘不变性关系

线性变换$T:R^3 \rightarrow R^2$，将一个三维向量（三维空间上的）映射成一个二维向量（二维空间上的）

这个线性变换对整个输入空间会产生什么影响？可以把输入空间用其一组基$v_1,v_2,\cdots,v_n$来表示，相应的输出空间也用其基$w_1,w_2,\cdots,w_m$表示

对于坐标的理解：向量由空间上的基线性表示，每个基的系数就是其在这个方向上的坐标值

研究线性变换对基向量的影响$T(v_1)=a_{11}w_1+a_{21}w_2+\cdots+a_{m1}w_m$$T(v_2)=a_{12}w_1+a_{22}w_2+\cdots+a_{m2}w_m$，将输入空间基用输出空间基的线性组合来表示，同时可以看出T应该是m*n维的，把n*1维向量变成了m*1维

矩阵逆、矩阵乘法，都可以解释成相应的线性变换操作

e.g. 求导、投影都是线性运算

行/列向量标准正交——矩阵的逆等于其转置

在信号图像应用中，很多情况下用小波基或者傅里叶基，但最好的基是特征向量基，因为此时转换矩阵M是特征值矩阵，不过找特征向量基代价较大。

Summary

  • 矩阵可对角化的本质是有足够的特征向量可以线性无关
  • 微分方程$\frac{du}{dt}=Au$，矩阵指数$e^{At}$
  • 投影矩阵的特征值为0和1
  • 对称矩阵，总有实特征值，并且有”足够”的特征向量。特征向量组成的矩阵Q是正交矩阵，有$A=Q\Lambda Q^T$
  • 正交矩阵的特征值得绝对值等于1，正交矩阵的作用就像旋转，不会改变向量的长度
  • 相似矩阵 $B=M^{-1}AM$，A和B总有相同的特征值。他们就是在不同的基下作用一样的矩阵。他们特征值相同，但特征向量一般不同。
  • 当矩阵满足：$AA^T=A^TA$ 时，A的特征向量正交。对称阵、反对称阵、正交矩阵是满足此条件的三类矩阵；
  • SVD：任意矩阵 $A=UΣV^T=(orthog)(diag)(orthog)$，V是$A^TA$的特征向量矩阵，U是$AA^T$的特征向量矩阵。当A是对称矩阵时，U=V。具体做奇异值分解时
    1. 奇异值σ是对角矩阵Σ对角线上的元素，它的值等于$A^TA$的特征值的正平方根
    2. 但值得注意的一点是：在求$AA^T$的特征向量的时候，方向要由$Av_i=σ_iu_i$来确定。

e.g.

$\frac{du}{dt}=Au=\left[ \begin{matrix} 0 & -1 & 0 \ 1 & 0 & -1 \ 0 & 1 & 0 \end{matrix} \right] $

该方程的通解形式是$u(t)=c_1e^{\lambda_1t}x_1+c_2e^{\lambda_2t}x_2+c_3e^{\lambda_3t}x_3$

  1. 求A的特征值

    第1、3行线性相关，A必有一个特征值为0；A是反对称矩阵，因此所有特征值都在虚轴，易求。

  2. 解的周期？

    e的虚数次方周期性的特征是：$e^{\sqrt{2}iT}=e^0$，即可求出T

  3. 验证A的特征向量是正交的。

    易证，因为对称矩阵、反对称矩阵、正交矩阵的特征向量总是正交的

  4. 解如何表示成$u(t)=e^{At}u(0)$？

    当$A=S\Lambda S^{-1}$时，$e^{At}=Se^{\Lambda t}S^{-1}$，$e^{\Lambda t}=diag(e^{\lambda_i t})$，实质上是求A的特征值和特征向量

$\lambda_1=0,\lambda_2=c,\lambda_3=2$

$x_1=[1 \quad 1 \quad 1]x_2=[1 \quad -1 \quad 0]x_3=[1 \quad 1 \quad -2]$

  1. c取何值矩阵可以对角化

    可以对角化意味着有3个线性无关的特征向量，题中3个向量不仅独立，还是正交的。所以：无论c取何值，都可以对角化。

  2. c取何值矩阵是对称的

    如果c是实数那么A是对称矩阵

  3. c取何值矩阵是正定、半正定的

    特征值里已经有0了，所以没法正定了，当c>=0时候，半正定

  4. 这个矩阵是Markov 矩阵么

    Markov矩阵有一个特征值为1，其余特征值<1，所以不可能

  5. $P=\frac{1}{2}A$是否是投影矩阵

    投影矩阵的特征值全为实数，并且为0和1，要求c为0或2

SVD分解：若一个矩阵SVD分解的$\sum=\left[ \begin{matrix} 3 & 0 \ 0 & 2 \end{matrix} \right]$，U和V都有两列

  1. A的性质如何

    $U,V,\sum$都是可逆的，所以A也可逆

  2. 如果$\sum=\left[ \begin{matrix} 3 & 0 \ 0 & -5 \end{matrix} \right]$

    不可能，$\sum$里的奇异值不能是负数

  3. 如果$\sum=\left[ \begin{matrix} 3 & 0 \ 0 & 0 \end{matrix} \right]$，奇异，秩为1

已知A对称且正交

  1. A的特征值是？

    1或-1。因为对称阵的特征值是实数，正交矩阵相乘不会改变向量的长度，其特征值绝对值是1。

  2. A是否正定？

    不一定，可能是-1，如$\left[ \begin{matrix} 1 & 0 \ 0 & -1 \end{matrix} \right]$

  3. A没有重复的特征值？

    错，A是3*3以上就一定有重复特征值（只能是1，-1）

  4. A可以对角化？

    对，所有对称和正交矩阵都可以对角化，其交集也可以

  5. A可逆？

    对，正交矩阵都可逆

  6. 证明$P=(A+I)/2$是投影矩阵

    验证$P^2=P$，其中用到$A^2=A^TA=I$另外，A的特征值为0和1，运算一下发现P的特征值也是1和0，符合投影矩阵要求
