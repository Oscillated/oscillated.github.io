32位简单汇编语言分析

代码段+分析

  • include和includelib的区别
    • include 用于预编译
    • includelib 用于生成.obj之后link操作连接成.exe
  • .data 是引导词，之后定义变量数据
  • 汇编命名变量与C有所不同，名称在前数据类型在后，100 dup(0) 对括注内容重复100次，相当于char result[100]={0}（剩下99元素会自动赋值为0），
    • dup可以嵌套 s db 3 dup(“ABC”)
  • 定义format变量的原因是输出函数需要格式声明作为一个输入（以字符串类型填入%d，再用于wsprintf）
    • char format[] = “%d”;
    • 汇编中不能采用C的\0结尾隐藏，要用逗号隔开加0的方式实现 -> ‘\0’, ascii=0的字符
  • char prompt[] = “The result”;
  • .code 是引导词，之后为代码部分
  • main: 是标号，程序第一条指令不一定是上下顺序的第一条，main可有可无，由结尾的end确定程序起始执行点
  • mov a, b
    • a = b;
    • eax: extended ax 寄存器
  • next: 也是标号，可用于jump操作
  • add a, b
    • a = a + b;
  • cmp a, b
    • compare a with b
  • jbe = jump if below or equal
    • b below
    • e equal
    • a above
  • invoke 函数调用的伪指令，编译后会变成call指令，有点像宏，因为call本身不能带参数所以用invoke和栈结构来实现参数调用
    • invoke 函数名 参数，参数，…
    • offset是取址，相当于&（要取数组的地址）
    • wsprintf(&result, &format, &eax)
      • 打印结果 使用格式 数据源寄存器
      • 编译分解为（将参数从右到左压入堆栈，再执行call指令）
        • push eax
        • push offset format
        • push offset result
        • call wsprintf
        • add esp, 0Ch
      • 压入三个4字节到栈，最后要释放 0Ch=12
    • MessageBox 弹框函数
      • 第一个0用于指定MessageBox父窗口的编号，即句柄，此处的编号必须为已经存在的
        • 父子关系的窗口存在联动性，移动父窗口时子窗口一并移动；销毁父窗口时，子窗口一并销毁。第一个参数设置为0，表示自己是一个独立的窗口，不存在父窗
      • result为弹框的正文，要取址，正文在先，标题在后
      • prompt为弹框的标题，是字符串数组要取址
      • 最后一个0表示弹框风格 最简单按钮0只有一个OK，其次有OK+CANCEL,YES+NO，也有左上角有×等
  • end
    • end后面的标号指定程序的起始执行点
    • 也就是刚开始eip值的代码行位置

16位简单汇编语言分析

  • 0Dh是回车符的ascii，0Ah是换行符的ascii
    • char upper = “Upper case\n”;
    • 表达光标从上一行末尾到达下一行行首
      • linux用\n
      • dos/windows用回车（光标移到同行行首）+换行（光标垂直下落一行）
  • 代码块有头有尾
    • name segment
    • name end
    • 代码块内有标号
  • assume 指定代码段和数据段
    • ds等于data段的段地址，data段内的变量地址均等于data段的段地址
  • mov中 段名表示该段的段地址
    • data 可用 seg abc表示
  • 做两次mov，原因是asm限制不能把一个常数赋给ds而只能从寄存器中赋值
  • 函数调用
    • mov ah, 1
      • 用于指定子函数的编号
    • int 21h
      • 调用21h函数集(dos)里的1号函数
    • int:interrupt中断，理解为调用
    • 相当于 al = getchar()
  • jae - jump if above or equal
    • 后跟一个地址或者标号
    • 多次cmp与jmp的组合构成多分支结构
    • 与C的明显差别：没有&,|符号
    • jnb - jump if not below
  • cmp
    • ax = ah + al
    • 16位 = 高8位 + 低8位
  • ah中断常用函数int 21h - refer to 终端大全
    • 1 getchar()
    • 9 输出字符串 dx必须为字符串地址
      • mov ah, 9
      • mov dx, offset str
        • 段地址+偏移地址
      • int 21h
    • 4Ch 程序终止
      • mov ah, 4Ch
      • mov al, 0
        • al是返回码，这行去掉也可以，返回码通常情况下无用，通常程序是并列关系，父亲为操作系统，而操作系统无需返回码
        • 返回码的用途：执行程序时a.exe中可以运行其他程序，例如b.exe，那么两程序如何实现通讯呢？父子程序 b.exe带回返回码，让a能够断定b成功与否
      • int 21h
      • 相当于exit(0);
