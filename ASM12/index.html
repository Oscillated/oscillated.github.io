字符串比较指令 cmpsb, cmpsw, cmpsd

cmpsb

比较byte ptr ds:[si]与byte ptr es:[di]
当DF=0时，SI++，DI++
当DF=1时，SI–，DI–

repe cmpsb

repeat if equal 若本次比较相等则继续比较下一个

先判断cx=0循环是否结束，再比较，之后不是马上判断，要先为下一次循环做好准备，再做判断

这样设定的结果是终止时多跳了一格，如果需要恢复就要减回去，但是还需要考虑因为循环次数结束（cx=0,两个字符正常结束）而结束比较的情况，因为如果循环次数结束，正常结束就不需要减回去

repne cmpsb

repeat if not equal 若本次比较不等则继续比较下一个

控制转移指令

jmp的3种类型

修饰变量：byte ptr, word ptr, dword ptr, fword ptr(修饰32位下的远指针，16位段+32位偏移), qword(64位) ptr, tbyte(80位) ptr,

修饰标号：near ptr, far ptr

一般情况下，编译器会自动度量跳跃的距离，因此我们在写源程序的时候不需要加上short、near ptr、far ptr等类型修饰。即上述三种写法一律可以简化为jmp target。

短跳指令的格式

jmp 偏移地址或标号
以下条件跳转指令也都属于短跳: jc jnc jo jno js jns jz jnz ja jb jae jbe jg jge jl jle jp jnp

若跳转距离可以用1字节表示，则此指令是短跳指令(-128-127,80h-7Fh)

计算机病毒会将自己转移到自己隐蔽的内存空间防止被杀掉

病毒当前在1000:0000运行

它把自己复制到9000:0000-9000:FFFF处

然后修改dos内核里面的一个变量MemSize -= 64K，标记9000:0000之后那一段是不存在的，杀毒软件无法到达这块空间

操作系统的启动也会涉及代码的移动，cpu在开机后从FFFF:0000执行第1条指令做上电自检 power on self test 读取硬盘0道0头1扇区到0000:7C00处，jmp 0000:7C00 执行硬盘中的代码(master boot record)，mbr移动自己到另外一块空间，然后读取C盘的第一个扇区到0000:7C00处，再jmp 0000:7C00
