乘法指令 mul

只有一个操作数寄存器 另一个被乘数寄存器是al/ax/eax

例如mul bh表示ax=al*bh

8位乘法

两个乘数分别是al和操作数寄存器，乘积在ax

16位乘法

两个乘数分别是ax和操作数寄存器，乘积在dx:ax，意思是乘积结果为dx*$2^{16}$+ax，因为结果是32位一个16位寄存器无法存储，需要保存在两个寄存器中

32位乘法

两个乘数分别是eax和操作数寄存器，乘积在edx:eax，意思是乘积结果为edx*$2^{16}$+eax

除法指令 div

16位除以8位得8位

AX / 除数 = AL … AH （被除数在ax，除数是操作数寄存器，商在al，余数在ah）

记忆：小端规则，al存商在先，ah存余数在后

设ax=123h, bh=10h

32位除以16位得16位

dx:ax / 除数 = ax … dx （被除数”段地址“在dx，”偏移地址“在ax，除数是操作数寄存器，商在ax，余数在dx）

设dx=123h, ax=4567h, bx=1000h

64位除以32位得32位

edx:eax / 除数 = eax … edx （被除数”段地址“在edx，”偏移地址“在eax，除数是操作数寄存器，商在eax，余数在edx）

假定要把一个32位整数如7FFFFFFFh转化成十进制格式，则一定要采用这种除法防止发生除法溢出。

远指针、近指针

远指针 far pointer

16位汇编中，远指针是指16位段地址+16位偏移地址;

32位汇编中，远指针是指16位段地址+32位偏移地址。

近指针 near pointer

16位汇编中，近指针是指16位的偏移地址；

32位汇编中，近指针是指32位的偏移地址；

远指针(far pointer)包括段地址及偏移地址两个部分；

近指针(near pointer)只包括偏移地址，不包含段地址。
