ZJU Computer Organization & Design By 刘鹏 6(lec09)

I-Format

是处理立即数的指令

在R型指令的形式上做出一些调整

缩短funct段，func7与rs2合并成12位field用于表示12位的一个立即数

opcode = 0010011 for all I-Format Immediate instructions

移位

I型指令还可用于移位指令，imm的12位field的末5位表示移位位数（0-31位），funct3和imm高七位合起来表示移位运算类型（funct3区分左移右移，imm第二位区分逻辑移位和算术移位）

Load

载入指令也是一种I型指令

imm代表偏移（4的倍数），rs1代表基址，载入的值存于rd，funct3规定载入值的长度（lb lh lw lbu lhu）。

opcode = 0000011 for all I-Format Load instructions

设x2为数组A，则以下代表x14=A[2](8/4=2)

S-Format

S型指令用于存储

高7位imm和低7-11的5位inm合成，代表12位偏移值，rs1代表基地址，rs2表示要被存储数据的寄存器编号，funct3规定载入值的长度（sb sh sw）。

S型指令不会向寄存器写入值（没有rd）

opcode = 0100011 for all S-Format instructions

设x2为数组A，则以下代表A[2]=x14(8/4=2)
