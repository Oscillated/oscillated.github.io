Scholar_Infer_Typing_Attacks

“学者云会面”讲座10

  • Traditional methods to record the keystrokes for an adversary were some installed software in the target computer.

  • Existing research work has identified a new class of attacks that can record the keystrokes in a non-invasive way without infecting the target computer to install software : Pressing a key of a keyboard can cause a unique and subtle environmental change, which can be captured and analyzed by the attackers to learn the keystrokes.

  • however, a training phase must be accomplished to establish the relationship between an observed environmental change and the action of pressing a specific key. This significantly limits the impact and practicality of these attacks.

  • it is possible to design keystroke eavesdropping attacks without requiring the training phase :

    • a joint demodulation algorithm : to establish a map-ping between typing each letter and its respective environmental change by exploiting the correlation among observed changes andknown structures of dictionary words.

  • bypass

    • the requirement of the training phase,which is impractical for most attack scenarios
    • the requirement to deceive the user or bypass the user’s anti-virus and firewall software to install malware on the target device
    • the requirement of line-of-sight between the attacker’s device and the keyboard.
