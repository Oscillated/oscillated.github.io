Ubuntu-Installation

上了小白的汇编以后win10装了VMware，以前知道能用来装一下虚拟机，之后linux也需要用到，这周课上完以后就试了一下，记录一下坑和经验。

选了Ubuntu

选乌邦图主要是因为入门的话教程和资料会比较多，

安装iso

装VMware就不说了，看了下小白提供的是永久有效的VMware 15.5 Pro，真香！

看了一圈以后选了18.04.4 LTS的版本，一开始憨憨去挂梯子下官网的版本，后来发现降速很严重，突然想到了去看一下镜像，最后找了浙大镜像的版本（惊了之前真不知道浙大也有开源镜像站），速度在1-2M/s左右。

下载完之后主要参照的是这篇博客，虚拟磁盘空间我设了机械的128G。

安装VMware Tools

装VMware Tools是为了实现客户机和主机的各种交互以及客户机的全屏等功能，还是很有必要的，安装也很简单，在VMware的虚拟机选项卡就能下载，之后会在虚拟机桌面生成一个压缩包，之后的操作见这篇经验，装的过程中也初步了解了一下linux的终端命令行风格。

初始优化和设置

简单列几条比较实用的优化策略。

  • 更换最优软件源
  • 装gnome的扩展
  • 选择舒服的主题和图标样式、壁纸等
  • tweaks工具
  • 装常用软件，如编译器、输入法、文字编辑软件等
  • 终端推荐上oh-my-zsh，用过都说香。
