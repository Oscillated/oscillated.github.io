环境搭建

Python是一门解释型语言，需要安装Python解释器，它是Python运行的内核；同时为了使写代码的界面不那么丑，还需要安装支持Python语言的编辑器/开发工具软件，常用的有Pycharm、VS Code、Vim、Sublime等等。各位可以凭自己喜好使用。

这次我们以Windows下的VS Code作为Python的代码环境，它是微软推出的一款免费编辑器，采用的是“轻量化+插件化”的设计思路，通过安装不同的插件，VS Code能胜任不同代码环境的开发工作。

  • 没有用Pycharm是因为我的Pycharm学生版过期用不了了，并且Pycharm体积比较大，对基本的语法学习来说有些多余。
  • Vim在类Unix系统中用得比较多，考虑到大家的电脑大多是Windows就没有用。
  • Sublime个人没有太多接触过。

安装Python

Python的安装，基本按照这篇Python参考安装教程 ，文章最后的虚拟环境可以不用跟着装。

  1. 在python官网下载Python3，需要装64位的，版本可以任选一个3.7.x或者3.8.x下载（Python3.9刚发布，可能会有小bug或者后续不兼容的问题）。如果是MacOS或者Linux系统，则需要安装对应系统的Python IDE。

  2. 下载后打开安装程序，注意勾选”Add Python 3.X to PATH”，它把Python解释器的路径加入系统环境变量，在后面VS Code会方便一些。安装时有一些选项，参考上面的安装教程链接里的说明图。

  3. 配置环境变量，照教程做一步步做就行。

  4. 检查安装是否成功，win+r，输入cmd打开命令行，敲python --version弹出版本号，基本就安装成功了。再敲python进入python环境，print("Hello, world!")，检查一下能否使用。退出环境的话，敲exit()；退出命令行，再敲一个exit。

安装VS Code

参照这篇VS Code参考安装教程，然后回到Python参考安装教程，装完VS Code部分。

  1. 打开VS Code，侧边栏的扩展（Extensions）点开后，搜索Python，安装Python扩展插件。

  2. 配置Python环境的时候需要注意在配置文件中把pythonPath设置好，VS Code依照这个路径将程序在解释器中运行。
