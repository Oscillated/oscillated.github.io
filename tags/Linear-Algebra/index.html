    矩阵对角化

    从$Ax=b$问题拓展到$Ax=\lambda x$.

    $Ax$看作矩阵$A$作用在向量$x$上的过程，强调矩阵是线性变换的思想

    所研究的特殊性质是在矩阵作用后仍与原向量在同一直线上的这样的非零向量和倍增值

    求特征值时式子$(A-\lambda I)x=0$对于非零向量$x$有$A-\lambda I$为奇异矩阵的推断，由此与先前引入的行列式概念衔接。有关行列式的理解，感觉勃勃的[这篇][https://zhuanlan.zhihu.com/p/19609459]说的挺好。

    引入迹trace的概念，这里提到了一个之前没有讲过的关系

    能够对角化的矩阵，特征值之和等于迹，特征值之积等于行列式。

    对于三角矩阵以上的结论则更加简单，特征值就是主对角元素。

    矩阵的对角化要求矩阵有n个线性无关的特征向量，特征向量是由特征值代回$(A-\lambda I)x=0$求得的$x$，通过将特征向量横排成矩阵可以进行矩阵对角化$A=S\Lambda S^{-1}$，S为横排的特征向量而$\Lambda$是对角的特征值矩阵。

    通过矩阵对角化可以很快计算矩阵的方幂，这里课上讲的是计算斐波那契数列的问题，之前确实不知道这个离散系统的问题还能通过矩阵特征值的知识来求解。

    具体方法是通过数列递推公式$F(n)=F(n-1)+F(n-2)$构造差分方程。

    令$u_k=[F_{k+1}, F_{k}]^T$

    联立$F(k+2)=F(k+1)+F(k)$，$F(k+1)=F(k+1)$

    那么$u_{k+1}=Au_k$，$A=[{1,1},{1,0}]$，进而可以算其特征值和特征向量，解决计算第100项这样的问题。以上的方法还可以用于高阶微分方程的矩阵表示和求解

    吉老爷子，yyds。

    4.6

    矩阵与微分方程

    由一阶线性微分方程解的形式可以看出，各个指数函数（由矩阵特征值确定）是互不相干的，他们的起点状态由初值给出，之后各走各路。特征值为负，最终收敛；特征值为正，最终发散；特征值为复数，关注其实部，在收敛发散的区别下还有相位改变（震荡）。

    2*2矩阵，由类似韦达定理的推导，其trace<0,det>0，则系统稳定

    原微分方程右侧是两个相互耦合的未知函数，矩阵对角化、提取特征值的过程可以看成是解耦

    矩阵与级数

    几何级数和指数级数同样适用于矩阵

    $e^{At} = I+At+\frac{(At)^2}{2}+\frac{(At)^3}{6}+\cdots$ 总收敛

    $(I-At)^{-1} = I + At+(At)^2 + (At)^3 + \cdots$ (At)的特征值小于1

    解耦：$\frac{du}{dt}=Au, u=Sv \rightarrow \frac{dv}{dt} = S^{-1}ASv = \Lambda v$

    利用指数级数，

    $e^{At} = S S^{-1}+S\Lambda S^{-1}t+\frac{1}{2}{S\Lambda ^2 S^{-1}t^2}+\frac{1}{6}{S\Lambda ^3 S^{-1}t^3}+\cdots $

    $e^{At} = S(I+\Lambda t+(\Lambda t)^2+\cdots)S^{-1} = Se^{\Lambda t}S^{-1}$ 对可对角化的矩阵成立（n个线性无关特征向量）

    $e^{\Lambda t} = diag(e^{\lambda_1 t},e^{\lambda_2 t},\cdots)$

    $u(t)=e^{At}u(0) = [x_1,x_2,\cdots,x_n]diag(e^{\lambda_1 t},e^{\lambda_2 t},\cdots)[C_1,C_2,\cdots,C_n]^T $$= C_1x_1e^{\lambda_1 t}+C_2x_2e^{\lambda_2 t} + \cdots$

    $e^{At}$收敛 -> 所有特征值的实部是负数 -> $\Lambda$位于复平面左半平面

    矩阵的幂收敛 -> 位于复平面单位圆内

    矩阵与随机过程

    马尔可夫矩阵

    从列向量的角度看

    马尔科夫矩阵满足

    • 所有元素大于等于0
    • 所有矩阵列相加等于1 -> 保证了其有一个特征值为1（因为$A-I$的列相加为0，说明行向量线性组合能得到0，因此一个行向量能被其他行向量表示，他们线性相关，说明$A-I$是奇异矩阵）

    由于矩阵与其转置的特征值相等，第二个条件改为列相加等于1也是成立的

    马尔科夫矩阵典型例子：两州的人口迁移与最终稳定人口数

    已知一个特征值为1，对于二阶矩阵可以通过迹和行列式值迅速获得另一个特征值。

    矩阵与傅里叶级数

    先前提到的标准正交基问题，基是两两正交的向量，定义正交为内积$q_1^{T}\bullet q_1$，可以把一个向量分解为各个基上的投影分量。

    求解某个基的分量，将其对应的基在等式两边做内积即可。此时对于$v=x_1q_1+x_2q_2+\cdots \rightarrow Qx=v$，有$x=Q^{-1}v=Q^Tv$ 来求各项分量。

    傅里叶级数是无穷维的，基变成了两两正交的函数，正交性对任意两个函数仍然成立

    $f(x) = a_0 + a_1cosx+b_1sinx+a_2cos2x+b_2sin2x+\cdots$，正交性定义为两个正交函数的积分，对于这里的三角函数，定义域为$[0,2\pi]$，同样可以将对应基对等式两边同时做内积来求出分量。

    4.7

    习题

    研究$A$为向量时的投影矩阵$P=\frac{aa^T}{a^Ta} \quad a=[2,1,2]^T$

    1. 3*3矩阵P的秩为1（先前知识）

    2. 有两个0特征值

    3. 另一个特征值为1

      • 由迹得出为1

      • 由$Pa=a$的意义去理解，自己投影到自己上还是自己，这一式子符合特征值的定义，进而得到特征向量就是a自己。

    4. 给一个向量序列$u_{k+1}=Pu_k \quad u_0=[9,9,0]^T$，求$u_k$。事实上，$u_k=u_1$，因为第一次计算相当于把$u_0$投影在$a$上而得到$u_1$，之后的向量不过是重复做投影到$a$上的操作。

      • 另外，如果P不是投影矩阵，把$u_0$分解成特征向量的形式来理解，$A^ku_0 = c_1\lambda_1^kx_1 + \cdots \quad$对投影矩阵，因为有两个0特征值，只剩下了特征值为1的项，多次投影是不变的

    行列式部分所引出的三对角线矩阵的递推公式$D_n = D_{n-1} - D_{n-2}$用于求解差分方程

    $\left[ \begin{matrix} D_n \ D_{n-1} \end{matrix} \right] = \left[ \begin{matrix} 1 & -1 \ 1 & 0 \end{matrix} \right]\left[ \begin{matrix} D_{n-1} \ D_{n-2} \end{matrix} \right] $

    求特征值特征向量，出现虚数$\frac{1\pm\sqrt{3}i}{2}$

    $A$的6次方两个特征值均为1，因为上面的复数特征值辐角为60°，$A^6=S\lambda^6S^{-1}=I$

    Insights

    线性变换

    • 左乘一个行向量/矩阵，所得结果相当于对原矩阵的行向量做线性组合。
    • 右乘一个列向量/矩阵，所得结果相当于对原矩阵的列向量做线性组合。

    理解这两句话是1-4节课的核心和整个线性代数课程体系最最重要的基础。

    解齐次和非齐次线性方程

    列空间

    $AX = b$ 的理解是：若有解，$A$经过右乘$X$，能找到线性组合（唯一或不唯一）表示出$b$，即b落在由线性组合定义的$A$的列空间中。

    零空间

    $AX = 0$ 的理解是：能够满足该等式方程的向量$X$在$A$的零空间中。

    四个子空间

    $m*n$的矩阵：

    • 列空间（$m$维空间的子空间）
    • 零空间（$n$维空间的子空间）
    • 行空间（$n$维空间的子空间）
    • 行空间的零空间（$m$维空间的子空间）。

    列空间与行空间的零空间正交，零空间与行空间正交。

    相互间的正交关系会在之后的投影中打通理解。

    以上是5-8课的主要内容。

    空间、矩阵、秩、维数、基

    • 秩是矩阵进行消元法后pivot的个数。
    • 列向量满秩，则列空间的基是线性无关且都在列空间内的$n$个$m$维向量，维数为秩$r$。
    • 行向量满秩，则行空间的基是线性无关且都在行空间内的$m$个$n$维向量，维数为秩$r$。
    • 零空间的维数是自由变量的个数$n-r$，满秩时零空间为零向量。
    • 矩阵对应秩，空间对应维数。

    以上是一些主干知识外的内容。

    投影矩阵与可逆阵的理解

    投影方程：$p = P*b$

    所有投影矩阵的性质：

    $P^T = P$

    $P^2 = P$

    $A$为可逆方阵（满秩） -> $A$的列空间为整个空间 -> 用$A$能够表示$n$维空间中的任意一个向量 -> 要求$b$在$A$的列空间上的投影（方程$Ax=b$） -> $A$的列空间能够任意表示所以可以直接表示出$b$ -> $b$的投影就是$b$本身不变 -> 投影矩阵$P=I$

    如果$A$不可逆，奇异，则找到的投影矩阵$P=A(A^TA)^{-1}A^T$，也可以剔除线性相关的向量，选列空间的基向量代入公式计算$P$。

    $A$为向量，$P=\frac{aa^T}{a^Ta}$

    投影矩阵的过程就是最小二乘法的原理，解$A^TA\hat{x} = A^Tb$
